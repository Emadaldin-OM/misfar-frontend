# Misfar – AI-Powered Smart Tourism Platform for Oman 🌍

Live Demo: https://misfar-frontend.onrender.com/

## About
Misfar is an AI-driven smart tourism web platform designed for Oman. 
Built as a graduation project and winner of:
- 🥇 1st Place – Best Innovative Idea Competition (2026)
- 🥇 1st Place – Startup Bootcamp Competition (2026)

## Features
- 🤖 LLM-powered chatbot for personalized travel recommendations
- 🗺️ Destination search with interactive maps and activities
- 📋 User profile dashboard (saved plans, liked destinations, comments)
- 🔔 User alerts system
- 🛠️ Full Admin Dashboard (manage destinations, users, activities)
- 📱 PWA support for offline use

## Tech Stack
- **Frontend:** React.js, Redux, PWA, Responsive Design
- **UI/UX:** Figma
- **Backend:** Node.js, Express.js (see misfar-backend repo)
- **Database:** PostgreSQL (Neon.tech)
